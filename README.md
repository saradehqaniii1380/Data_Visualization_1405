# Data Visualization Course Project – Spring 1405
## Project Objective

The objective of this project is to perform a comprehensive data visualization study on a real-world dataset and communicate insights effectively through visual analytics.

## Project Requirements
Each group must select an appropriate dataset and perform a complete data visualization analysis using Python.
## Repository Structure
Please organize your repository as follows:

```
GroupName/
│
├── data/
│   └── dataset.csv
│
├── notebook/
│   └── analysis.ipynb
│
├── report/
│   └── report.pdf
│
├── infographic/
│   └── infographic.pdf
│
└── README.md
```

## Submission Components

### 1. Dataset

Upload the original dataset used in the project.

Accepted formats:

* CSV
* XLSX
* JSON

Place the dataset inside the `data` folder.

### 2. Analysis Report

Prepare a PDF report including:

* Dataset description
* Data preprocessing
* Exploratory Data Analysis (EDA)
* Visualization methods
* Interpretation of findings
* Conclusions

Place the report inside the `report` folder.

### 3. Python Notebook

Upload a complete Jupyter Notebook containing:

* Data loading
* Data cleaning
* Exploratory analysis
* Visualizations
* Explanations and comments

Place the notebook inside the `notebook` folder.

### 4. Infographic

Design a one-page infographic summarizing the most important findings of your project.

Accepted formats:

* PDF
* PNG

Place the infographic inside the `infographic` folder.

## Minimum Visualization Requirements

Each project must contain at least:

* 2 Bar Charts
* 2 Line Charts
* 2 Distribution Plots
* 1 Correlation Visualization
* 1 Interactive Visualization
* Additional visualizations as needed
